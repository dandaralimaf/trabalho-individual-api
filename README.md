Trabalho de API com classe Funcionário e Endereço (com banco de dados H2 embutido)

## Como executar

1. Importe o projeto no STS
2. Rode a aplicação
3. Teste as rotas no Postman:

- GET /funcionarios
- GET /funcionarios/{id}
- POST /funcionarios
- PUT /funcionarios/{id}
- DELETE /funcionarios/{id}
